Menu Commands

File
New

Creates a new document in a separate tab for editing.

Open

Opens a saved document in a separate tab for editing.

Recent Files

Shows a list of recently opened documents for easy access. The "Clear History" menu item can be used to clear this list.

Close

Closes the currently active document. If it has unsaved changes the user is prompted to either save or discard the changes.

Save

Save the current document in the ISCII 7-bit encoding. The file name is asked only for newly created documents. Subsequent saves will simply overwrite the file.

Save As

Save the current document under a different name. The file name is asked every time.

Save Unicode Text As

Generates a unicode text file using the utf8 encoding that can be used with other unicode aware applications.

Save PDF As

Generates a PDF file that can be opened by Acrobat Reader or Foxit Reader.

Save RTF As

Generates a richtext document equivalent of the currently active document. The primary purpose of this is to allow text to be imported into word processors for typesetting local language scripts. NOTE: To be useful, the RTF file also needs the ISFOC fonts that are currently used by the editor.

Print Preview

Shows a preview of the currently active document.

Print

Print the currently active document. Pops up the system print dialog using which the user can set various options.

Exit

Closes the application. For each unsaved document, the user is prompted to either save or discard the changes.

Edit
Undo

Undoes the last editing operation. Currently, only a simple undo mechanism is available. It can undo only one deletion and that too only immediately after the deletion.

Cut

Removes the currently selected region of text and places it in the system clipboard and the internal clipboard.

Copy

Unlike cut, this command does not remove but only copies the currently selected region of text to the clipboards.

Paste

Inserts, at the current cursor position, the text that was sent to the internal clipboard by either the cut or copy commands. An interesting property of this command is that the text is always inserted in the currently active language. So, transliteration can be done by simply copying the text, changing the language and pasting it back.

Paste As ASCII

Inserts, at the current cursor position, the text that was sent to the system clipboard by not just this application but any other application. The text is always in the english language using the ASCII encoding. This command is useful for copying english data from other applications.

Clear

Removes the selected text from the document. The removed text cannot be pasted back.

Select All

A convenience command for selecting the entire document.

Select Block

A convenience command for selecting the current text block (paragraph).

Search
Find…

Activates the text finder just below the document area. This can be removed either by pressing the <ESC> key or by clicking the close button to the left of the find text box.

Find Next

Find the next occurrance of the word typed into the find text box starting the from the current cursor position.

Find Previous

Find the previous occurrance of the word typed into the find text box starting the from the current cursor position.

View
Show Keyboard

Displays an onscreen keyboard reflecting the currently selected key input method for the currently selected script.

Increase Text Size

Increase the text size of the active document by 2 points upon each invocation.

Decrease Text Size

Reduce the text size of the active document by 2 points upon each invocation.

UI Themes

A selection of widget appearance themes.

Show Statistics

When enabled, displays the word count of the currently active document.

Tools
Dictionary->Load

Prompt the user for a filename and load all words from the selected document into the dictionary. Currently disabled. The user can however open the document and index the document using <Ctrl>+<F8> shortcut.

Dictionary->Clear

Empty the contents of the dictionary. Note: once cleared, auto completion will not work untill the dictionary is loaded

Dictionary->Index Document

Load all the words of the document in the currently active tab into the dictionary. This is also activated by the shortcut <Ctrl>+<F8>.

Show Completions

Presents a list of known completions to the word typed upto the cursor. It can also be activated using the shortcut <Ctrl>+<Space>. <Escape> removes the completion list.

Help
Usage

Opens a new document tab and displays this file.

About

About this software.

About QT

About the QT Toolkit which this software uses.
